# InsightRadar 更新日志

## 2026-04-06

### 修复

#### 神策数据采集器
- **问题**: 只能采集到2023年的旧文章，无法获取2026年最新文章
- **原因**: 神策数据使用 Next.js 框架，文章通过 JavaScript 动态加载
- **解决**: 直接提取页面中的 `__NEXT_DATA__` JSON 数据
- **结果**: 成功采集到2026年最新文章（共416篇文章可用）

#### Amplitude 链接问题
- **问题**: "查看原文"链接返回 403 Forbidden 错误
- **原因**: 多个记录共享同一个列表页 URL
- **解决**: 
  - 优先使用文档链接（`/docs/`）
  - 改进文档链接提取逻辑，排除导航栏通用链接
  - 对于没有文档链接的记录，使用唯一的 release 详情页链接
- **结果**: 每个记录都有唯一、正确的链接

### 优化

#### 采集器架构
- 移除 Playwright 依赖，使用轻量级的 requests + BeautifulSoup
- 优化神策数据采集器，使用 JSON 解析替代 HTML 解析
- 改进 Amplitude 采集器的文档链接提取逻辑

#### 代码质量
- 修复 SQLAlchemy 保留字冲突（`metadata` → `task_metadata`）
- 修复 JavaScript 变量初始化顺序问题
- 清理项目中的临时文件和测试脚本
